应对高并发采取的措施：
                    1.分布式
                    2.页面静态化
                    3.使用Redis缓存
                    4.使用图片服务器
           #分布式
                    将一个系统进行物理分割，把分割后的子系统运行在不同的系统上
           #跨域访问
                    不同的网站(不同的ip地址或者端口)之间异步交换数据
                    + 解决方案：
                              1)设置header指令
                              2)jsonp:将数据包装为js代码
           #Redis
                    #功能：实现冷热分离
                    +热数据：
                            经常被访问的数据
                    使用步骤：
                            1)添加jsonp依赖
                            2)创建配置类，启用spring缓存
                            3)使用@Cacheable指定缓存的数据
                    实现冷热一致：
                            1)使用@CacheEvict重建缓存
